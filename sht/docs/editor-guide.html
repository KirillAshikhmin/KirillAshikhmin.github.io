Быстрый старт

  1. Нажмите «Открыть» и загрузите JSON или перетащите файл в окно редактора.
  2. Если файл содержит несколько шаблонов — выберите нужный в появившемся списке.
  3. Нажмите «Фикс в 1 клик» для автоматической коррекции и валидации.
  4. Скачайте результат и обновите шаблон на хабе.

Ручной режим

  1. «Исправить под требования Sprut.hub» — приводит структуру к актуальным требованиям.
  2. «Валидировать» — проверка по JSON Schema. Ошибки подсвечиваются со ссылками на узлы.
  3. «Форматировать» — автоформат JSON.
  4. «Скачать» — сохранение текущей версии шаблона.

Работа с архивами

Поддерживаются ZIP и TAR.GZ со множеством файлов. В диалоге выберите один из найденных шаблонов и продолжайте работу как обычно.

Мастер создания шаблона

Кнопка «Мастер создания» открывает пошаговый конструктор: выбор сервисов и характеристик, добавление привязок link, опций options, установка modelId и идентификации.
